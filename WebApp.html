Импорт в 1cData

Сначала загрузите базовую номенклатуру из 1С, затем подтяните текущую себестоимость и цену поставщика в уже созданную таблицу.

  1. Нажмите `Импорт из 1С номенклатуры`, выберите файл, сопоставьте колонки и выполните базовую загрузку.
  2. Проверьте, что табличная часть заполнилась и при необходимости откорректируйте `НДС`, `Пошлина` и `Цена поставщика` вручную.
  3. Нажмите `Импорт себестоимости текущей`, выберите файл и выполните обновление по номенклатуре, а если совпадения нет, то по артикулу.
  4. Нажмите `Импорт цены поставщика`, выберите файл и выполните обновление цен. Эта загрузка каждый раз перезаписывает колонку `Цена поставщика`.
  5. Для позиций типа `Наборы` отдельно используйте `Импорт комплектаций наборов`, чтобы загрузить составы наборов и затем выбрать активные спецификации на одноимённой вкладке.
  6. Используйте поиск и слайсеры над таблицей для проверки данных, затем сохраните ручные правки кнопкой `Сохранить ручные правки таблицы`.

Себестоимость флаконов и этикетки

Заполните ручные поля и используйте обновление для пересчёта вычисляемых колонок по данным импорта и параметрам *_fl.

  1. Загрузите номенклатуру, себестоимость и цену поставщика на вкладке Импорт данных.
  2. Откройте вкладку Флаконы и нажмите Обновить, чтобы собрать уникальные флаконы из 1cData.
  3. Цена поставщика, объём и вес подставляются маппингом по правилу Флакон = Наименование из вкладки Импорт данных.
  4. Проверьте и при необходимости скорректируйте НДС, пошлину и стоимость этикетки.
  5. Кнопка Пересчитать обновляет только вычисляемые колонки без перезагрузки страницы, а цветные колонки считаются по параметрам log_fl и com_fl.
  6. Нажмите Сохранить, чтобы записать ручные поля и текущие вычисленные значения на лист Флаконы.

Комплектации наборов

Импортируйте файл комплектаций, выберите активную спецификацию для каждого набора и задайте ручную стоимость только для тех компонентов, у которых нет найденной себестоимости.

  1. На вкладке Импорт данных сначала загрузите базовую номенклатуру, затем себестоимость 1С и цену поставщика.
  2. Нажмите `Импорт комплектаций наборов`, выберите файл отчёта и сопоставьте колонки `Компонент`, `Набор`, `Спецификация`, `Количество`.
  3. После импорта проверьте блок `Активные спецификации` и выберите по одному варианту для каждого набора.
  4. В блоке `Ручные стоимости компонентов` заполните повторяющиеся коробки, стикеры и наклейки, для которых ещё нет найденной себестоимости.
  5. Нажмите `Сохранить`, чтобы записать активные спецификации и ручные стоимости на лист `Наборы` и использовать их в расчёте себестоимости.
